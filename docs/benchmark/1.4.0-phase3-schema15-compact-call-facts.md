## Scope

This measurement validates the first independently committable Phase 3 optimization: schema 15 compact call facts. Repeated call text is interned in `call_strings`; entity and anchor digests are stored as 12-byte BLOBs; enum/provenance/boolean fields use checked integers and flags; call sites reference compact caller anchor IDs; persisted site fingerprints and expression UTF-16 coordinates are removed. The active read view reconstructs the existing typed rows at request time.

This change does not yet include deferred call indexes or side-by-side full-build publication. Those remain the next Phase 3 optimization because the write-path gate is not met.
